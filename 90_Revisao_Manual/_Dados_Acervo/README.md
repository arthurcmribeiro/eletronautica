# Dados do Acervo

Esta pasta e camada tecnica gerada por scripts, nao area editorial do vault.

Uso esperado:

- JSONs de apoio para pipeline, filas e auditorias;
- textos OCR brutos em `.txt`;
- notas OCR auxiliares geradas automaticamente;
- sidecars de texto para indexacao e rastreabilidade.

Regra editorial:

- nao usar estes arquivos como conteudo final;
- nao transformar OCR bruto em nota didatica sem curadoria humana;
- nao contar esta pasta como acervo legivel, trilha de curso ou material publicavel;
- promover apenas notas companheiras em `90_Revisao_Manual/_Acervo_Notas/` quando houver valor tecnico real.

Em resumo: `_Dados_Acervo` serve a automacao; o conteudo util para leitura fica fora daqui.
