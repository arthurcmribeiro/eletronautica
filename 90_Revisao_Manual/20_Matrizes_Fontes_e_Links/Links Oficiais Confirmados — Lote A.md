# Links Oficiais Confirmados — Lote A

> [!abstract] Resumo técnico
> Esta nota consolida os links oficiais já confirmados para o `Lote A` do backlog de coleta de manuais. Nesta rodada, `manual localizado` significa que já foi identificada pelo menos uma fonte oficial específica de manual, download técnico ou página de produto com seção dedicada de manuais/downloads. Isso ainda não significa que o PDF já foi baixado para o acervo local.

## Critério desta rodada

- priorizar páginas oficiais do fabricante;
- usar link direto de manual quando disponível;
- quando o fabricante centraliza tudo em uma página de produto com aba `Downloads` ou `Manuals`, usar essa página como âncora oficial;
- quando uma família tem várias SKUs, aceitar uma âncora por família para acelerar a coleta operacional;
- registrar lacunas explícitas quando um submodelo legado ainda não estiver isolado com link próprio nesta rodada.

## Observações práticas para a próxima passada

- baixar primeiro os PDFs das linhas `A01` a `A10`, porque energia costuma render mais valor diagnóstico por documento;
- na Garmin, priorizar PDFs de `742`, `VHF 215`, `GMI 20`, `GPS 19x HVS`, `Reactor 40` e `GMR 18 HD`, deixando modelos muito próximos para deduplicação depois;
- na Raymarine, a navegação mais segura continua sendo via páginas de `Manuals` por família;
- no Furuno, as páginas de suporte por SKU já estão suficientemente boas para baixar operador, instalação, template e software;
- no Icom, a central de `Manual Download` é excelente, mas alguns handhelds pedem busca por modelo mais novo (`IC-M25 EVO`) em vez do legado puro `IC-M25`.
