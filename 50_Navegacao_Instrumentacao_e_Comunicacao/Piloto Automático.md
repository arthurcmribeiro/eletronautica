# Piloto Automático

> [!abstract] Resumo técnico
> O **piloto automático (autopilot / heading control system — HCS / track control system — TCS)** é o sistema de controle em malha fechada que **mantém rumo (heading) ou rota (track)** atuando sobre o leme da embarcação, liberando o timoneiro para vigilância visual e demais tarefas — **sem nunca dispensar a vigília exigida por COLREGS Rule 5 + Rule 7**. Compõe-se de quatro blocos: **(1) sensor de heading** (vide [[Bússola Eletrônica (Compass - HDG Sensor)]] — fluxgate / AHRS / GPS compass), **(2) computador de controle** com algoritmo PID + sea-state + rudder gain, **(3) atuador de leme** (linear, rotativo, hidráulico ou bomba reversível) e **(4) interface humana** (display / remoto / integração com MFD). Padrões: **IMO MSC.64(67)/MSC.74(69)** (HCS performance), **IEC 62065** (TCS), **ISO 11674/16329** (HCS), **ABYC P-22/P-24/P-27** (steering systems). Modos típicos: **Heading** (rumo magnético/verdadeiro fixo), **Track** (segue rota GPS — XTE compensado), **Wind** (mantém ângulo de vento — veleiros), **Dodge** (desvio temporário com retorno automático), **Standby** (manual). Fabricantes dominantes: **Raymarine Evolution EV-100/200/300/400, Garmin Reactor 40 / GHP Reactor, Simrad AP44/AP70/AP80, Furuno NavPilot 711C/300, B&G H5000 Pilot, ComNav P-Series**.

> [!tldr] TL;DR — 10 regras inegociáveis
> 1. **Piloto automático NÃO substitui o vigia.** COLREGS Rule 5 + Rule 7 + IMO Resolution A.694(17) exigem vigia permanente. Autopilot mantém rumo, NÃO desvia de obstáculo, outra embarcação, mau tempo. Confiança cega = colisão garantida.
> 2. **Sensor de heading é o componente crítico.** Fluxgate sensível a interferência magnética (motor, alto-falante, inversor, alternador) causa "caça" de rumo. AHRS é referência atual em mar agitado; GPS compass é imune a magnético mas requer linha de vista. Vide [[Bússola Eletrônica (Compass - HDG Sensor)]].
> 3. **Calibração obrigatória após instalação:** swinging compass + dockside test do atuador + rudder gain + counter rudder + sea state. Sem isso, performance é inaceitável e/ou inseguro. Procedimento é POR ESCRITO no manual de bordo.
> 4. **Dimensionar atuador para ≥1,5× o torque de pico do leme** (ABYC P-22/P-24/P-27). Em ondas / mar agitado / vento forte, força no leme pode chegar a 8-12× o nominal de cruzeiro. Atuador subdimensionado trava (corrente >100% nominal sustained → motor queima).
> 5. **Cabo de alimentação e fusível dimensionados para corrente de PICO** (não nominal). Picos de 30-100 A em barcos médios; queda de tensão <3% sob carga (ABYC E-11.4.4.1.b). Fusível ANL/MIDI a 125% pico, ≤7" da bateria.
> 6. **Atuador hidráulico exige fluido específico** (ATF Dexron III/VI ou ISO VG 32 HLP — vide [[Óleos Hidráulicos Marine — Guia Completo]]). Fluido errado em sistema Octopus / Hynautic / ComNav rompe vedação em meses. Manual do fabricante é referência absoluta.
> 7. **Disjuntor / chave de emergência ACESSÍVEL E SINALIZADO** — botão "STANDBY" no helm + breakers separados nos dois pólos da bateria do atuador (ABYC P-24.5). Tripulação treinada para acionar em <2 segundos (ex.: rumo de colisão).
> 8. **Rudder gain (sea state gain)** ajustado para condições reais — não deixar no default de fábrica. Mar calmo: sea state alto (low gain). Mar agitado: sea state alto (low correção) para evitar "caça". Mar moderado: sea state médio.
> 9. **Track mode (rota GPS) consome mais corrente** que heading mode — XTE compensa corrente/vento → leme em movimento contínuo → atuador sob duty cycle alto. Em mar agitado, preferir **Heading mode + intervenção manual periódica**.
> 10. **MOB (Man Overboard) sempre força o piloto a STANDBY** — em queda d'água, manobra de retorno é manual. Se piloto continuar em rota, vítima é deixada para trás. Treinar tripulação no procedimento.

> [!danger] Cenários de risco
> - **Colisão por confiança cega no piloto:** noite + tripulação dorme + piloto em track mode + outra embarcação cruza a rota → impacto direto. **Caso histórico:** múltiplos casos documentados pelo USCG / IIMS Casualty Database. **Prevenção:** vigia permanente (revezamento); radar com alarme MARPA; AIS com CPA/TCPA alarms; treinamento.
> - **"Caça" do rumo em mar agitado** (oscilação de ±5°-30°): sensor fluxgate + ondulação induz ruído de heading + ganho mal ajustado → piloto corrige excessivamente → desgaste hidráulico/atuador + tripulação enjoada + risco em estreito. **Prevenção:** AHRS; sea state alto; rudder gain reduzido; manobra manual em condição limite.
> - **Atuador trava em condição de pico:** torque do leme excede capacidade do atuador (ondas, encalhamento parcial, redes de pesca) → corrente de stall 200-500% nominal → fusível abre OU motor queima → barco sem leme automático em momento crítico. **Prevenção:** dimensionar atuador 1,5-2× o pico esperado; treinar manual override.
> - **Falha total elétrica do piloto** (curto, fusível aberto, perda de comunicação NMEA): piloto desliga sem aviso → barco "sai do rumo" gradualmente em offshore noturno → grounding ou colisão. **Prevenção:** alarme audível de "piloto inativo"; fusível dedicado; redundância NMEA backbone com terminadores OK.
> - **Sensor de heading perturbado por nova carga elétrica** (instalou inversor próximo, novo som, etc.) → desvio sistemático de 5°-30° → barco desvia gradualmente do canal navegável → grounding em banco de areia. **Prevenção:** swinging após qualquer modificação elétrica significativa; documentar deviation.
> - **Atuador hidráulico com fluido errado:** sistema Octopus exigia ATF Dexron III, instalador colocou óleo de motor → vedações degradam em meses → vazamento + perda de pressão → leme escapa → manobra perigosa. **Prevenção:** consultar manual do atuador; etiqueta no reservatório; vide [[Óleos Hidráulicos Marine — Guia Completo]].
> - **Rope / linha de pesca / rede enrolada na hélice** + leme parcialmente bloqueado: atuador trava em alto torque → fusível abre → piloto desativa em momento crítico (manobra de aproximação). **Prevenção:** inspeção visual da água; alarme de stall; chave de emergência manual.
> - **Surto atmosférico via mastro queima HCS + AHRS + atuador:** raio próximo → tensão induzida em cabo de heading → eletrônica destruída → barco sem orientação automática. **Prevenção:** ABYC TE-04 surge protection; bonding mastro-quilha; backup com bússola magnética.
> - **MOB com piloto não desligado:** vítima cai → tripulação reage → piloto continua em rota → vítima fica metros atrás em segundos → resgate dificílimo. **Prevenção:** treinamento — primeiro ato no MOB é STANDBY do piloto + manobra de retorno (Williamson turn).
> - **Track mode com waypoint distante:** piloto recebe rota com waypoint a 50 nm + corrente lateral 2 nós → barco devia 2 nm da linha → autopilot tenta recuperar continuamente, lutando contra corrente → consumo elétrico alto + atuador desgastado + chegada errada. **Prevenção:** waypoints intermediários a cada 5-10 nm; monitorar XTE e corrigir manualmente.

## O que é (definição rigorosa)

O **piloto automático** é um sistema de **controle em malha fechada (closed-loop control)** que regula a posição do leme da embarcação para manter uma referência (heading ou track), comparando dado medido (sensor de heading) com dado desejado (setpoint) e aplicando correção via atuador.

Arquitetura geral:

```
                        [Setpoint: heading ou track]
                                 │
                                 ▼
[Sensor heading]──[Erro]──[Controlador PID]──[Driver]──[Atuador]──[Leme]──[Casco/dinâmica]
        ▲                                                                          │
        │                                                                          │
        └──────────────────────────────[Realimentação]─────────────────────────────┘
```

Quatro blocos principais:

### 1. Sensor de heading (vide [[Bússola Eletrônica (Compass - HDG Sensor)]])

- **Fluxgate compass** (entry-level): ISO 22090-2.
- **AHRS** (Attitude and Heading Reference System) — magnetômetro + acelerômetro + giroscópio MEMS.
- **GPS compass dual-antenna** — ISO 22090-3 — heading verdadeiro independente de magnético.

### 2. Computador de controle

Algoritmo PID + ajustes específicos do piloto náutico:

- **Proportional (P):** correção proporcional ao erro de heading.
- **Integral (I):** correção acumulada — elimina erro estacionário.
- **Derivative (D):** correção antecipativa — reduz overshoot.
- **Rudder gain:** ganho do leme (quanto leme aplica por grau de erro).
- **Counter rudder:** quanto contra-leme aplica para evitar overshoot.
- **Sea state:** filtro de ondulação — alto = menos correção em mar agitado.
- **Auto-tune:** algoritmo proprietário que ajusta ganhos automaticamente (Raymarine AutoLearn, Garmin Auto-Tune, Simrad Sea State Adapt).

### 3. Atuador

Quatro famílias:

#### Atuador linear (push-pull rod)

```
[Computador]──[Driver]──[Motor DC]──[Redutor]──[Parafuso de esferas]──[Haste]──[Braço do leme]
```

- Aplicação: barcos <50 ft, leme com braço acessível.
- Modelos: Raymarine Type 1/2/3, Lewmar Mamba (legacy), Simrad MD-Series.
- Vantagens: instalação simples, custo médio.
- Desvantagens: força limitada, espaço para o atuador.

#### Atuador rotativo (acoplado à coluna do volante)

```
[Computador]──[Driver]──[Motor DC]──[Redutor]──[Acoplamento à coluna do volante]
```

- Aplicação: barcos <40 ft com volante mecânico.
- Modelos: Simrad RPU80/RPU200, Raymarine WS Type 1.
- Vantagens: instalação no console, sem modificar leme.
- Desvantagens: força limitada, depende de volante mecânico.

#### Bomba hidráulica reversível (drives hydraulic steering)

```
[Computador]──[Driver]──[Motor DC]──[Bomba hidráulica]──[Cilindro do leme existente]
```

- Aplicação: barcos com direção hidráulica de volante (Octopus, Hynautic, Capilano).
- Modelos: Octopus 12V/24V, Hynautic Hydraulic Pump, ComNav 1500AP.
- Vantagens: integra-se ao sistema hidráulico existente; alta força.
- Desvantagens: requer fluido hidráulico correto; manutenção do sistema hidráulico.

#### Drive-by-wire (eletrônico nativo)

```
[Computador]──[CAN bus / NMEA 2000]──[ECU do steering]──[Atuador integrado]
```

- Aplicação: motoryachts modernos (Volvo Penta IPS, Mercury Joystick, ZF SteerCommand).
- Modelos: Garmin Reactor Steer-by-Wire, Raymarine SmartPilot.
- Vantagens: integração total, força via servo motors do barco.
- Desvantagens: requer barco com sistema fly-by-wire compatível.

### 4. Interface humana

- **Display dedicado** (Raymarine p70R, Garmin GHC 20, Simrad AP44).
- **MFD touchscreen** (controle pelo chartplotter).
- **Remoto wireless** (S100 Sirius, Garmin GHC 50 Wireless).
- **Joystick** (em sistemas IPS / pod drive).

## Modos de operação (típicos em pilotos modernos)

| Modo | Função | Quando usar |
|------|--------|-------------|
| **Standby** | Piloto desativado, controle manual | Sempre em manobra de aproximação, manobra de pesca, MOB |
| **Heading (Auto)** | Mantém rumo magnético/verdadeiro fixo | Cruzeiro em mar aberto, sem rota planejada |
| **Track / Nav** | Segue rota GPS — compensa XTE | Rota planejada com waypoints |
| **Wind (vela)** | Mantém ângulo de vento (TWA/AWA) | Veleiros — vela contra o vento, vela em popa |
| **Dodge** | Desvio temporário (manual) com retorno automático | Desviar de obstáculo flutuante mantendo a rota |
| **Pattern (pesca)** | Padrões pré-definidos (zig-zag, círculo, search) | Pesca de arrasto, busca SAR |
| **Power Steer** | Joystick controla taxa de giro do leme | Manobra de aproximação a píer |
| **No Drift** | Mantém posição (com bow thruster) | Mega-iate, charter, ancoragem dinâmica |

## Calibração — procedimento estruturado

### Etapa 1 — Pré-instalação (dockside)

1. **Verificar atuador** sob carga manual (não elétrica): sem bind, curso completo, lubrificação.
2. **Confirmar dimensionamento** — leme aplica ~50% do torque com volante manual?
3. **Cabos elétricos** dimensionados para corrente de pico (datasheet do atuador).

### Etapa 2 — Swinging do sensor de heading

Procedimento ISO 22090-2 (vide [[Bússola Eletrônica (Compass - HDG Sensor)]]):

1. **Águas calmas, longe de píer metálico**.
2. **Girar 360°** suavemente em ~2 minutos (3°/s).
3. **Aguardar erro residual <2°**.
4. **Documentar** deviation em 8 rumos cardinais.

### Etapa 3 — Dockside test do atuador

1. **Comando manual** de leme via interface — verificar movimento na direção correta.
2. **Tempo lock-to-lock** — comparar com spec do fabricante.
3. **Sem batente** — atuador deve parar antes do limite mecânico do leme.

### Etapa 4 — Sea trial — calibração de PID/sea state/rudder gain

1. **Velocidade típica de cruzeiro**, mar moderado.
2. **Modo Heading** com setpoint estável.
3. **Iniciar Auto-tune / AutoLearn** se disponível (Raymarine, Garmin).
4. **Manualmente:** ajustar **rudder gain** até não haver "caça" mas o piloto não derivar.
5. **Counter rudder:** ajustar para reduzir overshoot.
6. **Sea state:** ajustar conforme condição (alto em mar agitado).

### Etapa 5 — Track mode test

1. **Configurar rota** com 3-5 waypoints.
2. **Ativar Track**.
3. **Monitorar XTE** — não deve passar 50-100 m em mar moderado.
4. **Ajustar Track gain** se XTE excessivo.

### Etapa 6 — Wind mode test (veleiros)

1. **Vento estável** 8-15 nós.
2. **Modo Wind** com TWA/AWA setpoint.
3. **Verificar** que o leme corrige automaticamente para manter o ângulo.

## Falhas mais comuns (com causa raiz e diagnóstico)

| # | Falha | Causa raiz típica | Diagnóstico |
|---|-------|-------------------|-------------|
| 1 | "Caça" do rumo (oscilação) | Rudder gain alto / fluxgate com interferência | Reduzir gain; trocar para AHRS; swinging |
| 2 | Deriva gradual de rumo | Rudder gain baixo / sensor mal calibrado | Aumentar gain; re-swing |
| 3 | Atuador trava | Sólido na hélice / batente do leme | Inspeção visual; verificar limite mecânico |
| 4 | Piloto desliga sozinho | Queda de tensão sob pico / fusível dimensionado mal | Multímetro com motor em RPM; re-fusível |
| 5 | Atuador hidráulico vaza | Fluido errado / vedação degradada | Consultar manual do fabricante; trocar selo |
| 6 | "No heading data" | Cabo NMEA aberto / sensor falhou | Software monitor NMEA; substituir cabo/sensor |
| 7 | Track XTE alto sustentado | Corrente forte / track gain baixo | Aumentar track gain; waypoints intermediários |
| 8 | MOB ativado mas piloto continua | Configuração de MOB não fez force standby | Verificar config MOB; treinar tripulação |
| 9 | Piloto liga mas não move o leme | Fusível atuador aberto / driver morto | Multímetro alimentação atuador; substituir driver |
| 10 | Wind mode "caça" o vento | TWA jitter > rudder gain limit | Filtro de wind smoothing; reduzir wind gain |

## Diagnóstico profissional (procedimento estruturado)

### Etapa 1 — Levantamento

1. Modelo do piloto + sensor + atuador, ano de instalação, última calibração.
2. Sintoma exato (caça / deriva / desliga / atuador trava).
3. Histórico (mar agitado, manobra recente, modificação elétrica).

### Etapa 2 — Verificação elétrica (com piloto em standby + atuador em alimentação)

1. Tensão nominal nos terminais do computador (≥12,5 V em 12 V).
2. Tensão sob acionamento do atuador (queda <3%).
3. Corrente nominal sob acionamento típico (verificar contra spec).
4. Corrente em stall (= se atuador travado intencionalmente — não exceder 30 s).
5. Continuidade de NMEA backbone (60 Ω entre CAN-H e CAN-L).

### Etapa 3 — Verificação do sensor de heading

(vide [[Bússola Eletrônica (Compass - HDG Sensor)]] — etapas 1-3)

### Etapa 4 — Verificação mecânica do atuador

1. **Linear:** curso completo, sem bind, parafuso lubrificado.
2. **Rotativo:** rotação livre na coluna, sem batente.
3. **Hidráulico:** nível do fluido, ausência de vazamento, fluido correto, ar removido.
4. **Drive-by-wire:** códigos de erro do barramento CAN.

### Etapa 5 — Sea trial diagnóstico

1. Ativar piloto em modo Heading com setpoint estável.
2. Verificar se barco mantém rumo dentro de ±2° em mar moderado.
3. Observar comportamento do leme (frequência, amplitude).
4. Ajustar gain/sea state e re-testar.

### Etapa 6 — Decisão

- Calibração mal feita → re-calibrar.
- Sensor degradado → upgrade fluxgate → AHRS.
- Atuador desgastado → kit de revisão ou substituir.
- Computador com falha → substituir.
- Cabos / NMEA → reparar.

## Boas práticas operacionais

- **Vigia permanente** — COLREGS Rule 5 não é negociável.
- **Treinamento** da tripulação em manual override + STANDBY <2 s.
- **Calibração** após instalação + qualquer modificação elétrica + anual.
- **Inspeção semestral** do atuador (lubrificação, fluido, conexões).
- **Backup** de heading (bússola magnética + AHRS redundante em offshore).
- **Alarmes audíveis** ativados (off-course, low voltage, atuador stall).
- **Documentar** ganhos calibrados no manual de bordo.
- **MOB protocol** — configurar para forçar STANDBY do piloto.
- **Surge protection** na alimentação (ABYC TE-04).
- **Bonding** do atuador e computador (ABYC E-11.16).

## Erros comuns (e por que são erros)

- "Piloto automático dispensa o vigia." → Falso. COLREGS Rule 5 exige vigia mesmo com piloto.
- "Não preciso calibrar — sai de fábrica calibrado." → Falso. Cada barco tem dinâmica única (peso, leme, casco) — calibração é específica.
- "Sea state alto sempre é melhor." → Falso. Em mar calmo, sea state alto causa deriva (low gain). Ajustar dinamicamente.
- "Track mode é melhor que Heading sempre." → Falso. Track consome mais corrente; em mar agitado, Heading com intervenção manual é mais confiável.
- "Atuador hidráulico aceita qualquer ATF." → Falso. Sistema Octopus exige Dexron específico; outros usam ISO VG 32 HLP. Manual é referência.
- "Piloto desliga sozinho — equipamento ruim." → Não necessariamente. Queda de tensão por cabo subdimensionado é causa mais comum.
- "MOB não muda nada se piloto está bom." → Erro grave. MOB exige STANDBY imediato; piloto continuando deixa vítima para trás.

## Glossário (termos técnicos)

- **Autopilot:** piloto automático.
- **HCS (Heading Control System):** sistema que mantém heading.
- **TCS (Track Control System):** sistema que mantém rota (com XTE).
- **Heading:** direção da proa.
- **Track:** rota planejada (linha entre waypoints).
- **XTE (Cross-Track Error):** distância perpendicular entre barco e rota.
- **PID (Proportional-Integral-Derivative):** algoritmo de controle clássico.
- **Rudder gain:** ganho do leme (graus de leme por grau de erro).
- **Counter rudder:** contra-leme antecipativo.
- **Sea state:** filtro de ondulação no algoritmo.
- **Auto-tune / AutoLearn / Auto-Adapt:** algoritmo proprietário de calibração automática.
- **Atuador linear:** atuador push-pull.
- **Atuador rotativo:** atuador acoplado à coluna do volante.
- **Bomba reversível:** atuador hidráulico (bomba DC reversível).
- **Drive-by-wire (DBW):** controle eletrônico nativo (sem cabo mecânico).
- **Steer-by-wire:** sinônimo de DBW para steering.
- **Standby:** modo manual.
- **Heading mode (Auto):** mantém rumo fixo.
- **Track mode (Nav):** segue rota GPS.
- **Wind mode:** mantém ângulo de vento (veleiros).
- **Dodge:** desvio temporário com retorno automático.
- **Pattern mode:** padrões pré-definidos (zig-zag, círculo).
- **No Drift:** mantém posição (com bow thruster).
- **MOB (Man Overboard):** queda d'água — força STANDBY.
- **Williamson turn:** manobra de retorno padrão para MOB.
- **CPA (Closest Point of Approach):** distância mínima de aproximação.
- **TCPA (Time to CPA):** tempo até CPA.
- **MARPA:** Mini Automatic Radar Plotting Aid.
- **Stall current:** corrente de rotor bloqueado (alta).
- **Lock-to-lock:** tempo total de leme de batente a batente.
- **Rudder Reference Sensor:** sensor de posição do leme.
- **PGN 127245:** Rudder (NMEA 2000).
- **PGN 127237:** Heading/Track Control (NMEA 2000).
- **NMEA 0183 $APB:** Autopilot sentence.
- **NMEA 0183 $RMB:** Recommended Minimum Navigation.
- **NMEA 0183 $XTE:** Cross-Track Error.
- **NMEA 0183 $HSC:** Heading Steering Command.
- **NMEA 0183 $RSA:** Rudder Sensor Angle.
- **IMO MSC.64(67):** primeira norma IMO de HCS performance.
- **IMO MSC.74(69):** norma revisada de HCS.
- **IEC 62065:** TCS performance.
- **ISO 11674 / ISO 16329:** HCS para HSC e ships.
- **SOLAS V/24:** uso de HCS/TCS.
- **COLREGS Rule 5:** Look-out (vigia).
- **COLREGS Rule 7:** Risk of collision.
- **AHRS:** Attitude and Heading Reference System.
- **Fluxgate:** sensor magnético de heading.
- **GPS compass:** sensor GNSS dual-antenna.
- **Octopus:** marca de atuador hidráulico marine.
- **Hynautic / Capilano:** marcas de direção hidráulica marine.
- **ATF Dexron:** fluido de transmissão automotiva (alguns atuadores).
- **ISO VG 32 HLP:** óleo hidráulico mineral (vide [[Óleos Hidráulicos Marine — Guia Completo]]).
